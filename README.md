# Air Hockey

## Background and Overview

Air Hockey is a 2d game. The goal is to get the puck past the opponent's paddle.

## Functionality & MVP

Users will be able to:
 - [ ] Play against another user.
 - [ ] Play against an AI.
 - [ ] Choose the speed of the puck.
 - [ ] Add multiple pucks
